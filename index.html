User manual

What every tool does, in one glance.

Edit PDFRewrite existing text, add formatted text boxes (font, size, bold/italic/underline, alignment, any colour), draw, highlight, erase, whiteout, add movable shapes, ticks, crosses, stamps and images — then save a real PDF.
Edit existing textIn the editor, choose the text-cursor tool and click any line of text to rewrite it in place. The original font, weight, colour and size are matched automatically. Works on digital PDFs (not scans).
WhiteoutA brush that paints with the page's own background colour — adjust its size with the stroke slider and rub over anything, exactly like an eraser. A separate WO box tool drags a rectangle for covering larger areas.
Dark & light modeUse the moon/sun button in the header to switch themes. Your choice is remembered on this device.
Organize pagesReorder, rotate, duplicate or delete pages. Insert blank pages, other PDFs or images anywhere.
MergeCombine every open PDF into one file. Reorder with the arrows on each file.
SplitExtract a page range, or burst the document into one PDF per page.
WatermarkStamp text or a logo across every page, with adjustable opacity and size.
Page numbersAdd numbering in the position and format you choose.
CompressShrink file size — smart, lossless or maximum, with a before/after size comparison.
Rotate allTurn every page 90°, 180° or 270° in one click.
SignaturesDraw a signature, type your name in a Brush Script handwriting style, or import a photo of your signature — the background is removed and the signature is isolated from surrounding text automatically. Pick any ink colour, optionally add a timestamp below the signature ("Name (Jun 15, 2026 10:49:09 EDT)"), and save as many named signatures as you like.
Ticks & crossesCheck and cross stamps drop as movable objects — drag, resize, recolour, duplicate or delete them just like shapes.
PDF to WordExport pages into an editable .doc that opens in Word.
PDF to PowerPointEach page becomes a full-bleed slide in a .pptx deck.
PDF to ExcelPulls text line-by-line into a spreadsheet you can clean up.
PDF to JPGRenders every page as a high-quality image, zipped per document.
Images to PDFTurn a set of photos or scans into a single PDF.
Excel to PDFThe first sheet of each workbook becomes a tidy PDF table.
Word to PDFExtracts the text of a .docx and typesets it into a PDF.
Nothing is uploaded. All processing happens locally in your browser using WebAssembly and JavaScript. Close the tab and everything is gone.
